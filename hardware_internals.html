Hardware Internals

Motherboard
The Motherboard is the main board (or system board) that has physical parts where the electronic circuits provide communication among the parts which are being connected. The following parts will be explained below.
1. Central Processing Unit (CPU)
2. Basic Input/Output System (BIOS)
3. Memory
4. Mass storage interfaces
5. Serial and Parallel ports
6. Expansion slots
7. Controllers
8. ROM
9. RAM
10. Cache
11. Graphics
